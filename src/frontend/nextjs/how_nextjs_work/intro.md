在创建第一个 Next.js App 之前，先来看看 Next.js 是如何工作的，与 React 项目相比，它有哪些不同？我将从以下几个方面来探讨：

- **代码运行环境**：**开发环境 Dev** or **生产环境 Prod**；
- **代码运行阶段**：**构建时 Build Time** vs **运行时 Runtime**；
- **在哪儿渲染**：**客户端 Client** vs **服务端 Server**。

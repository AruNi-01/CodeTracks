    1. 什么是 AQS

    AQS 全称 AbstractQueuedSynchronizer，即抽象队列同步器，是 JUC 包下的一个 构建锁和同步器的框架。

    AQS 是一个抽象类，内部提供了一些使用 protected 修饰的方法，意图就是让我们自己 继承 AQS，重写这些模板方法，即可来构建自己的同步器。
